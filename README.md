# E-Commerce Customer Churn Analysis (MySQL)

This project analyzes e-commerce customer data using MySQL to identify churn patterns and key factors affecting customer retention. The insights help drive data-driven decisions for improving customer loyalty.

## Table of Contents
- [Project Overview](#project-overview)
- [Data Description](#data-description)
- [Analysis Approach](#analysis-approach)
- [Results & Insights](#results--insights)

## Project Overview
**Goal:** Identify customer churn patterns and the main drivers of churn in an e-commerce setting using SQL-based analysis.

## Data Description
- **Dataset:** “Synthetic dataset representing 10,000 online shoppers from 2022.”
- **Main columns:**  
  - `CustomerID`, `OrderDate`, `LastPurchaseDate`, `TotalOrders`, `ChurnStatus`, etc.

## Analysis Approach
- Performed exploratory data analysis, aggregation, segmentation.
- Identified key churn factors: recency, frequency, monetary value, etc.

## Results & Insights
- Customers with infrequent purchases are 2.5x more likely to churn.
- Promotional email engagement reduces churn by 15%.
